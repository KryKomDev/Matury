# Čtyřúhelník

- 4 vrcholy, 4 strany
- konvexní (všechny úhle $< 180°$), nekonvexní (alespoň jeden úhel $> 180°$)

## Rovnoběžníky

- protější strany jsou rovnoběžné
- protější úhlý jsou stejné
- úhlopříčky se půlí

### Kosodelník

- $a \neq b \land \alpha \neq \beta$
- $\alpha + \beta = \gamma + \delta = 180°$
- $O = 2a + 2b$
- $S = a \cdot v_a$

### Kosočtverec

- $a = b \land \alpha \neq \beta$
- $\alpha + \beta = \gamma + \delta = 180°$
- $O = 4a$
- $S = a \cdot v_a$
- úhlopříčky jsou kolmé a půlí se

### Obdélník

- $a \neq b, \alpha = \beta = \gamma = \delta = 90°$
- $O = 2a + 2b$
- $S = a \cdot b$
- úhlopříčky jsou kolmé a půlí se

### Čtverec

- $a = b, \alpha = \beta = \gamma = \delta = 90°$
- $O = 4a$
- $S = a^2$
- úhlopříčky jsou kolmé a půlí se

## Lichoběžníky

- dvě, různě dlouhé rovnoběžné základny
- dvě ramena
- $\alpha + \delta = \beta + \gamma = 180°$
- $O = a + b + c + d$
- speciální typy:
  - pravoúhlý
  - rovnoramenný

## Deltoid

- složen ze dvou rovnoramenných trojúhelníků se společnou základnou ($BD$)
- $|BC| = |DC| \land |AD| = |AB|$
- úhlopříčky jsou kolmé, ale nepůlí se
